Notifier

Examples

The notifier allows you to display and manage notifications for your application.

Basic Modal

Toggle a modal via JavaScript by clicking the button below. It will slide down and fade in from the top of the page.

Click on the buttons to see an example.

{% endhighlight %} {% highlight js %} // somewhere in your application, instantiate the notifier. var notifier = new MarionetteComponents.Notifier(); $('[data-action="notification"]').on('click', function(event){ var button = $(event.target); var type = button.data('action-type'); var notification = { type: type, title: 'Hey!', message: 'What a nice notification!' }; // The notifier listens to the global event bus of Backbone Backbone.trigger('notifier:notify', notification); // But you can achieve the same thing with this: notifier.trigger('notifier:notify', notification); // Or notifier.notify(notification); // You can also use the helper functions Backbone.trigger('notifier:info', notification); // sets type = 'info' Backbone.trigger('notifier:warning', notification); // sets type = 'warning' // ... you get the point. }); {% endhighlight %}
